# PythonExpenseTracker
Key Features:
- Smart Categorization: Automatically distinguishes between one-time purchases and recurring bills.
- Instant Insights: Generates summary reports totaling spending by category and month.
- Error Prevention: Uses pattern matching (Regex) to ensure dates are entered correctly.
- Data Persistence: Saves your information to a local file so you never lose your progress.
- Quality Assured: Includes automated tests to ensure the math and logic are always accurate.

How to Run: 
1. Downloand the files
2. Open your terminal in the folder
3. Run "python expense_tracker.py"
